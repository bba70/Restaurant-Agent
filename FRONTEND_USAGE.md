# 前端界面使用指南

## 快速开始

### 1. 启动后端服务

```bash
python main.py
```

服务将在 `http://localhost:9000` 启动

### 2. 打开前端界面

直接在浏览器中打开 `index.html` 文件：

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

或者在浏览器地址栏输入：
```
file:///d:/desktop/Food/index.html
```

### 3. 开始使用

在搜索框中输入你的查询，例如：
- "我在北京想吃川菜，价格100元以下"
- "帮我推荐一下陕西科技大学附近的美食"
- "上海浦东新区的日料餐厅"

点击"搜索推荐"按钮或按 Enter 键进行搜索。

## 常见问题

### Q: 页面打不开或连接失败？

A: 检查以下几点：
1. 后端服务是否正在运行（`python main.py`）
2. 后端服务是否在 `http://localhost:9000`
3. 浏览器控制台是否有错误信息（F12 打开开发者工具）

### Q: 搜索没有结果？

A: 可能原因：
1. API Key 无效或配额不足
2. 城市名称不被识别
3. 网络连接问题

### Q: 如何修改后端地址？

A: 编辑 `index.html`，找到第 192 行：
```javascript
const API_URL = 'http://localhost:9000';
```

改为你的后端地址，例如：
```javascript
const API_URL = 'http://192.168.1.100:9000';
```

### Q: 如何部署到生产环境？

A: 
1. 将 `index.html` 放在 Web 服务器（如 Nginx）
2. 配置 CORS 和反向代理
3. 更新 `API_URL` 为生产环境地址

## 自定义样式

### 修改主题色

编辑 CSS 中的颜色值：
```css
/* 主题色：紫色 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 改为蓝色 */
background: linear-gradient(135deg, #2196F3 0%, #1976D2 100%);

/* 改为绿色 */
background: linear-gradient(135deg, #4CAF50 0%, #388E3C 100%);
```

### 修改卡片布局

编辑 CSS 中的网格设置：
```css
.restaurants {
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
}

/* 改为 2 列 */
grid-template-columns: repeat(2, 1fr);

/* 改为 1 列 */
grid-template-columns: 1fr;
```
